## Setup

  ##### Clone the repository
```bash
git clone https://github.com/Vigneshwaran-Mani/ERP.git
```
  ##### Move to the desired folder
```bash
cd \ERP
```
  ##### To install the dependencies, simply write
```bash
npm install
```

  ##### To run the app, simply write
```bash
npm start
```
